# 🎮 איזה סוג שחקן אתה? · שאלון מודל ברטל

שאלון אינטראקטיבי בעברית (RTL) שמגלה את **טיפוס השחקן** שלך לפי **מודל ברטל** —
הכובש, החוקר, החברותי או הלוחם — עם דיוקני פיקסל-ארט אישיים וסיכום מפורט.

מותאם לקהל חינוכי / נוער.

## 🕹️ הדגמה חיה
👉 **https://lemons-gaming.github.io/Bartle/**

## ✨ מה יש בו
- **12 שאלות תרחיש** בטון נגיש, עם ערבוב סדר התשובות.
- **4 דיוקני פיקסל-ארט** באיכות גבוהה (16-bit, פלטה מוגבלת) — מוטמעים כ-base64, ללא תלות רשת.
- **סיכום עשיר**: טיפוס ראשי + משני, פסי התפלגות באחוזים, נקודות חוזק וטיפ מותאם.
- עיצוב RPG/קלף, פונטים מסוגננים (Suez One + Rubik), עובד גם במובייל.

## 🚀 הרצה
פשוט פותחים את `index.html` בדפדפן (דאבל-קליק) — הכול עצמאי, ללא התקנה.

## 📁 מבנה
| קובץ | תיאור |
|---|---|
| `index.html` | האפליקציה העצמאית (React + Tailwind דרך CDN) |
| `מודל ברטל - שאלון שחקן.tsx` | קומפוננטת React לשילוב בפרויקט |
| `questions-data.js` | 12 השאלות + תוכן התוצאות |
| `result-logic.js` | חישוב הציון (דירוג, אחוזים, טיפוס ראשי/משני) |
| `art-data.js` | האומנות מוטמעת כ-base64 |
| `assets/portraits/` | קובצי הפיקסל-ארט הסופיים (PNG) |
| `aseprite/`, `_build/` | סקריפטים לייצור האומנות ולבניית הקבצים |

## 🛠️ בנייה מחדש
לאחר עריכת תוכן (`questions-data.js` / `result-logic.js`) או אומנות:
```bash
python3 _build/build.py
```
פעולה זו מרכיבה מחדש את `index.html` ואת קובץ ה-`.tsx` עם הנתונים המעודכנים.

## 🎨 צינור האומנות
תהליך היברידי: רפרנס ב-AI (Gemini) → הורדת רזולוציה ל-128px → המרה לפלטה מוגבלת ב-**Aseprite** → רקע אחיד.
